# ReadWise Offline

## What This Is

PWA-приложение для чтения саммари нон-фикшн книг на русском языке. Пользователи читают текстовые саммари, слушают аудиоверсии и просматривают карточки ключевых идей. Ключевое преимущество — полноценный офлайн-доступ через скачивание контента на устройство. Контент создается вручную автором проекта и загружается через админ-форму.

## Core Value

Доступные саммари нон-фикшн книг на русском языке с полным офлайн-доступом — дешевле конкурентов, удобнее в использовании.

- OAuth (Google, GitHub) — email/пароль достаточно для V1
- Видео-контент — сложность хранения, не в фокусе

## Context

- Целевая аудитория: русскоязычные предприниматели, менеджеры, студенты 25-45 лет
- Конкуренты: Smart Reading (дороже), Blinkist/Shortform (только на английском)
- Ценовое преимущество: 200 руб/мес vs ~400-500 руб/мес у Smart Reading
- Контент создаётся вручную автором — нет AI-генерации
- Приложение сгенерировано через Lovable, дорабатывается вручную
- Текущий каталог пуст (тестовые данные) — потребуется наполнение 30-50 саммари для запуска
- Существующий код имеет архитектурные паттерны: React Query для серверного состояния, Context API для клиентского, localforage для офлайна

## Constraints

- **Tech Stack**: React + TypeScript + Vite + Tailwind + shadcn/ui + Supabase — менять нельзя, кодовая база уже на этом стеке
- **Payments**: YooKassa (ЮKassa) — единственный доступный процессинг для РФ физлиц
- **Language**: Весь UI и контент на русском языке
- **Storage**: Supabase Storage для аудио (до 100 МБ на файл), офлайн-хранилище до 500 МБ на устройство
- **Budget**: Минимальные затраты на инфраструктуру — Supabase free/pro tier, Vercel бесплатный хостинг
