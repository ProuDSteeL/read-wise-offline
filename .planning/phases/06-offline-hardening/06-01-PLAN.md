**Then implement (GREEN):**

1. Create `src/components/OfflineBanner.tsx`:

```typescript
import { WifiOff } from "lucide-react";
import { useOnlineStatus } from "@/hooks/useOnlineStatus";

const OfflineBanner = () => {
  const isOnline = useOnlineStatus();
  if (isOnline) return null;

  return (
    <div className="fixed top-0 left-0 right-0 z-[60] flex items-center justify-center gap-2 bg-destructive/90 px-4 py-2 text-xs font-medium text-destructive-foreground">
      <WifiOff className="h-3.5 w-3.5" />
      Нет подключения к интернету
    </div>
  );
};

export default OfflineBanner;
```

Key design decisions (from research):
- `fixed top-0` with `z-[60]` -- above sticky headers (z-20) and other overlays
- `bg-destructive/90` -- semi-transparent red, attention-worthy
- Small text (`text-xs`) -- informational, not intrusive
- No dismiss button -- banner auto-hides when connectivity returns
- Russian text matching the app's language

2. In `src/App.tsx`, import and render `OfflineBanner` alongside `GlobalAudioPlayer`:

```typescript
import OfflineBanner from "./components/OfflineBanner";

// In the App component, after GlobalAudioPlayer:
<AppRoutes />
<GlobalAudioPlayer />
<OfflineBanner />
```

The OfflineBanner uses `useOnlineStatus()` internally so it works in both the online and offline route sets. When offline, it renders on top of the offline-only routes (DownloadsPage, OfflineReaderPage). When online, it returns null.
