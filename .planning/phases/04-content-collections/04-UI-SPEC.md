## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (gap-1) |
| sm | 8px | Compact element spacing, card internal gaps, carousel dot gaps (gap-2) |
| md | 16px | Default element spacing, section padding (px-4, p-4) |
| lg | 24px | Section vertical spacing (space-y-6) |
| xl | 32px | Layout gaps between major homepage sections |
| 2xl | 48px | Page-level vertical padding |
| 3xl | 64px | Not used in this phase |
| 4xl | 112px | MiniAudioPlayer clearance (pb-28) -- component is 80px tall + 32px breathing room |

Exceptions: none -- all values are multiples of 4.

## Typography

| Role | Size | Weight | Line Height | Usage in Phase 4 |
|------|------|--------|-------------|-------------------|
| Label | 11px (text-[11px]) | 400 (normal) | 1.4 | Stat card labels ("Прочитано", "Время", "Серия дней") |
| Body | 14px (text-sm) | 400 (normal) | 1.5 (leading-relaxed) | Key idea card content, continue card metadata, stat labels |
| Subheading | 16px (text-base) | 600 (semibold) | 1.3 (leading-snug) | Key idea card titles, section headers (SectionHeader component) |
| Display | 20px (text-xl) | 600 (semibold) | 1.3 (leading-snug) | Stat card values (existing ProfilePage pattern) |

Notes:
- Font weights used: 400 (body/labels) and 600 (headings/titles/stats) -- exactly 2 weights
- Section headers (previously 17px bold) consolidated to 16px semibold -- 1px difference was imperceptible
- Stat card values (previously text-xl font-extrabold 800) changed to text-xl font-semibold 600 for weight scale consistency
- All typography tokens already established in prior phases, no new tokens introduced

## Color

| Role | Value (HSL) | Usage |
|------|-------------|-------|
| Dominant (60%) | `--background: 38 60% 97%` (warm cream) | Page backgrounds, carousel background |
| Secondary (30%) | `--card: 38 50% 99%` (off-white) | ContinueCard, stat cards, collection book cards |
| Accent (10%) | `--accent: 157 18% 62%` (sage green) | Dot indicator active state, stat card icons, "See all" link text |
| Primary | `--primary: 350 78% 21%` (burgundy) | Key idea card gradient background, progress bar fill, active dot indicator |
| Destructive | `--destructive: 0 72% 51%` (red) | Not used in this phase (no destructive actions) |

Accent reserved for:
- Active carousel dot indicator (w-5 bg-primary elongated pill)
- Stat card icons (BookOpen, Clock, Flame) via `text-sage`
- "See all" link text in SectionHeader via `text-sage`
- Progress bar fill in ContinueCard via `bg-primary`

Key idea card color: Uses `gradient-primary` class (burgundy gradient) with `text-primary-foreground` (cream text). This is an existing pattern -- class may need definition if missing. Fallback: `bg-primary text-primary-foreground`.

## Interaction Contracts

### Key Ideas Carousel (CONT-01)

| Property | Value |
|----------|-------|
| Gesture | Horizontal swipe (touch) or drag (mouse) |
| Slides visible | 1 at a time (full-width, `basis-full` CarouselItem default) |
| Snap behavior | Embla default snap-to-slide |
| Navigation | Dot indicators below carousel (no arrow buttons on mobile) |
| Dot indicator: inactive | `w-1.5 h-1.5 rounded-full bg-muted-foreground/20` |
| Dot indicator: active | `w-5 h-1.5 rounded-full bg-primary transition-all` |
| Dot container | `mt-3 flex justify-center gap-2` |
| State sync | Embla `onSelect` callback updates `activeIdx` via `api.selectedScrollSnap()` |
| Animation | Embla defaults (no custom easing) |
| Empty state | Hide entire key ideas section (no carousel rendered) |

### Horizontal Book Rows (COLL-02)

| Property | Value |
|----------|-------|
| Scroll | `overflow-x-auto scrollbar-hide` (existing pattern) |
| Gap between cards | `gap-3` (12px) |
| Container padding | `px-4` horizontal padding matching page margin |
| Bottom padding | `pb-2` for scroll shadow clearance |
| Empty state | Hide entire collection section if no books assigned |

### Continue Reading Section (CONT-03)

| Property | Value |
|----------|-------|
| State | Already implemented -- verify only |
| Tap target | Full ContinueCard is tappable (button element) |
| Feedback | `tap-highlight` utility (scale 0.97, opacity 0.80) |
| Empty state | Hide section if no books with 0 < progress < 100% |

### Reading Stats (CONT-04)

| Property | Value |
|----------|-------|
| State | Already implemented -- verify only |
| Layout | 3-column grid (`grid grid-cols-3 gap-3`) |
| Card style | `rounded-2xl bg-card p-4 shadow-card` |
| Icon color | `text-sage` (sage green) |
| Zero state | Show "0" values for new users (always visible) |
