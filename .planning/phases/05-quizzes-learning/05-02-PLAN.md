Single question with 4 tappable option cards. Props: `{ question: { id: string; question: string; option_a: string; option_b: string; option_c: string; option_d: string; correct_option: number }; selectedAnswer: number | null; onAnswer: (optionIndex: number) => void }`.

- Default state: `bg-card border-border`
- When `selectedAnswer` is null (no answer yet), tapping calls `onAnswer(index)`
- When `selectedAnswer` is set (feedback mode):
  - Correct answer (index === question.correct_option): `bg-green-50 border-green-500` + Check icon in green-600
  - Wrong selected (index === selectedAnswer && index !== correct_option): `bg-red-50 border-red-500` + X icon in red-600

  - Other unselected: `bg-card border-border` (neutral)
- When answer is selected, show feedback text above options: "Верно!" (green-600 font-bold) or "Неверно" (destructive font-bold)
- Disable pointer events on all options during feedback (`pointer-events-none`)

State management -- quiz state machine (local useState):
```typescript
type QuizState =
  | { phase: "ready" }
  | { phase: "in-progress"; currentIndex: number; answers: Record<number, number> }
  | { phase: "completed"; score: number; answers: Record<number, number> };
```

Implementation:
- `useParams<{ id: string }>()` to get bookId
- `useBook(id!)` for book title in header
- `useQuiz(id!)` for quiz questions
- `useFlashcards(id!)` for flashcard data
- `useFlashcardProgress(id!)` for mastery data
- `useSaveQuizResult()` mutation
- Header: sticky top bar with `ArrowLeft` button navigating back (`navigate(-1)`) and book title truncated
- Tabs using shadcn `Tabs` defaultValue="quiz":
  - `TabsList className="grid w-full grid-cols-2"`
  - `TabsTrigger value="quiz"` label "Квиз"
  - `TabsTrigger value="flashcards"` label "Карточки"

**Quiz tab content:**
- If no questions (empty or loading): show empty state card with "Квиз пока не добавлен" heading and "Квиз появится здесь, когда автор добавит вопросы к этой книге" body
- If `quizState.phase === "ready"`: show "Начать тест" primary button + question count badge ("{questions.length} вопросов")
- If `quizState.phase === "in-progress"`:
  - `QuizProgress` component with total/current/answers
  - `QuizQuestion` for current question
  - `selectedAnswer` state resets to null on each new question
  - On answer: set `selectedAnswer`, after 1200ms setTimeout advance to next question or complete
  - Score calculation: count answers where `answers[i] === questions[i].correct_option`
- If `quizState.phase === "completed"`:
  - Call `useSaveQuizResult` on enter (useEffect)
  - `QuizResults` with score, questions, answers, onRetake (reset to ready), onBack (navigate back)

**Flashcard tab content:**
- If no flashcards: show empty state card with "Карточек пока нет" heading and "Карточки появятся здесь, когда автор добавит их к этой книге" body
- Otherwise render `FlashCardDeck` with flashcards data, bookId, and progress data

**Error state:** If useQuiz or useFlashcards errors, show "Не удалось загрузить данные. Попробуйте обновить страницу"

Page wrapper: `className="animate-fade-in pb-28"` consistent with other pages.
