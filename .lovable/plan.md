

# Саммари книг — PWA приложение

## Обзор
PWA для чтения саммари нон-фикшн книг на русском языке. Редакционный минималистичный дизайн в стиле "Тихая библиотека" — серифный шрифт Literata для чтения, керамические поверхности, фиолетово-зелёный акцент.

---

## Фаза 1: Фундамент и дизайн-система

### Дизайн-система
- Палитра: Deep Iris (#6C5CE7), Sage Green (#00B894), керамический белый фон
- Шрифты: Inter (UI/заголовки) + Literata (ридер)
- Компоненты: BookCard, KeyIdeaCard, AudioPlayer, MiniPlayer
- Анимации: stagger 40ms, scale 0.98 на тап, cubic-bezier(0.32, 0.72, 0, 1)
- Тёмная и сепия темы

### PWA-настройка
- vite-plugin-pwa с манифестом и иконками
- Service Worker для кеширования

### Навигация
- Нижняя панель: Главная, Поиск, Полки, Профиль
- Контейнер max-w-md по центру (мобильный PWA-формат)

---

## Фаза 2: База данных и аутентификация (Supabase)

### Таблицы
- **books** — title, author, cover_url, description, why_read (jsonb), about_author, categories, tags, read_time_min, listen_time_min, views_count, rating, status (draft/published/archived)
- **summaries** — book_id, content (markdown), audio_url, audio_size_bytes
- **key_ideas** — book_id, order_index, title, content
- **user_roles** — user_id, role (admin/user) — отдельная таблица с Security Definer функцией
- **profiles** — user_id, name, avatar_url, subscription_type, subscription_expires_at
- **user_progress** — user_id, book_id, progress_percent, last_position, audio_position
- **user_shelves** — user_id, book_id, shelf_type (favorite/read/want_to_read)
- **user_highlights** — user_id, book_id, text, note
- **user_downloads** — user_id, book_id, content_type, size_bytes
- **collections** — title, description, cover_url, book_ids, is_featured, order_index

### Аутентификация
- Email/password регистрация и вход
- Профиль создаётся автоматически через триггер
- RLS-политики на все таблицы

### Storage
- Бакеты: book-covers, audio-files, avatars

---

## Фаза 3: Главная страница и каталог

### Главная
- Баннер-карусель с акциями/подборками (swipeable)
- «Продолжить чтение» — карточки с прогресс-линией
- Подборки по категориям (горизонтальные списки)
- Новинки и популярное
- Персональные рекомендации (на основе user_progress и user_shelves)

### Каталог и поиск
- Полнотекстовый поиск (Supabase full-text search) по названию, автору, тегам
- Chips-фильтры: категория, время чтения, рейтинг, новизна
- Сетка карточек: обложка + название + автор + время чтения

---

## Фаза 4: Страница саммари

- Обложка (3:4, shadow-xl) → мета-информация → теги
- Блок «О книге» + «Зачем читать?» (3–5 пунктов)
- «Об авторе» — биография
- «Ключевые мысли» — горизонтальная карусель карточек (bg-primary, белый текст)
- Кнопка «Скачать» с прогресс-кольцом
- «С этим саммари читают» — похожие книги
- Sticky-футер: [ Слушать ] (ghost) и [ Читать ] (solid)
- Превью для Free-пользователей, полный доступ для Pro или купивших

---

## Фаза 5: Ридер

- Чистый текст Literata 18px, line-height 1.7
- Прогресс-бар 2px сверху (не скруглённый)
- Тап по центру — переключение UI (оглавление, настройки)
- Настройки: размер шрифта, тема (светлая/тёмная/сепия), межстрочный интервал
- Long press — выделение текста → сохранение цитаты
- Оглавление с быстрой навигацией по разделам
- Автосохранение прогресса в user_progress
- Баннер «Вы читаете офлайн» при отсутствии сети

---

## Фаза 6: Аудиоплеер

- Полноэкранный плеер: большая кнопка play/pause (64px), обложка
- Скорость 0.5x–2.0x (tabular-nums, без layout shift)
- Перемотка ±15 сек, таймер сна
- Мини-плеер sticky внизу при навигации по приложению
- Синхронизация позиции в user_progress
- Кнопка скачивания аудио

---

## Фаза 7: Офлайн-режим

- Service Worker перехватывает запросы, отдаёт кеш
- Текст → Cache API, аудио → IndexedDB (localForage)
- Кнопка «Скачать» → прогресс-кольцо → бейдж «Офлайн»
- Раздел «Мои загрузки» — список, размер, удаление
- Десатурация UI при потере сети, opacity-50 для нескачанных
- Автоудаление при превышении лимита (500 МБ по умолчанию)

---

## Фаза 8: Профиль и полки

- «Мои полки»: Избранное, Прочитано, Хочу прочитать
- «Мои загрузки» с управлением хранилищем
- Статистика: прочитано книг, общее время, серии
- Сохранённые цитаты и заметки
- Управление подпиской

---

## Фаза 9: Админ-форма загрузки контента

- Защищена ролью admin (проверка через has_role)
- Форма книги: название, автор, обложка, категории/теги, описание, «зачем читать», об авторе
- Загрузка саммари: Markdown-редактор или файл, предпросмотр в стиле ридера, авторасчёт времени чтения
- Загрузка аудио: MP3/M4A до 100 МБ, автоопределение длительности
- Ключевые идеи: 5–10 карточек с drag-and-drop сортировкой, предпросмотр карусели
- Статусы: Черновик / Опубликовано / Архив

## Фаза 10: Монетизация

- Freemium: 5 полных саммари бесплатно, превью всех
- Подписка Pro (299₽/мес, 1990₽/год) — безлимит + аудио + офлайн
- Покупка отдельного саммари (99–149₽) — доступ навсегда
- Интеграция ЮKassa (или Stripe)
- Таблица payments для истории транзакций
